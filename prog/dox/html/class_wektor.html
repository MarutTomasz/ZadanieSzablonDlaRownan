template<class TYP, unsigned int ROZMIAR>

Szablon klasy Wektor jest parametryzowany typem przechowywanych zmiennych oraz rozmiarem (ilością elementów).

Parametry
TYP- typ zmiennych tworzących wektor.
ROZMIAR- ilość zmiennych tworzących wektor.
